# Angular Address Book Application

## Overview

This project is an Angular Address Book application developed using Angular and DevExtreme DataGrid.

The application supports:

- Create Contact
- Read Contact List
- Update Contact
- Delete Contact

## Features

### Contact Information

Each contact contains:

- Name
- Email
- Phone Number
- Street
- State
- Postcode
- Description

### CRUD Operations

- Add new contact through popup modal
- View contacts in DevExtreme DataGrid
- Edit existing contact
- Delete contact with confirmation dialog

### Validation

- Name is required
- Email must be valid format
- Phone must contain numbers only
- Postcode must contain numbers only

## Technologies Used

- Angular 22
- TypeScript
- DevExtreme Angular Components
- DevExtreme DataGrid
- DevExtreme Popup
